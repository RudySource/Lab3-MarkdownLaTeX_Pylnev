#### Блоки кода

Встроенный (inline) код:
Команда **`git status`** показывает состояние репозитория.
Многострочный код:

```python
print("Hello, Markdown!")
```

Можно указать язык: python, js, bash, csharp и др.

#### Цитаты

Цитата создаётся символом >:

> Это цитата в Markdown.
> Вложенная цитата:
> Первый уровень
>
> > Второй уровень
