## Установка 3x-ui

### Шаг 1. Обновление системы

```bash
apt update && apt upgrade -y
```

### Шаг 2. Установка панели

```bash
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

### Шаг 3. Запуск сервиса

```bash
systemctl start x-ui
systemctl enable x-ui
```

---

## Основные команды

| Команда      |    Назначение    |
| :----------- | :--------------: |
| x-ui start   |  Запуск панели   |
| x-ui stop    | Остановка панели |
| x-ui restart |    Перезапуск    |

## Формула нагрузки:

Нагрузка сервера условно растёт как $L \sim n^2$, где _n_ — число клиентов[^1].
[^1]: Тест Latex в Md.

$$
\sum_{i=1}^n i = \frac{n(n+1)}{2}
$$
